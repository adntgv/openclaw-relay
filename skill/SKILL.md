# OpenClaw Relay Skill (Scaffold)

## Purpose
Provide OpenClaw commands to deploy/manage a relay and connect clients.

## Commands (planned)
- `relay init` — deploy relay server (Coolify or Docker)
- `relay token create` — issue client token
- `relay client install` — output install steps
- `relay status` — list connected clients

## Files
- `scripts/relay.sh` — placeholder CLI entry
- `templates/` — Coolify or Docker templates

## Notes
This is scaffolding only. Implementations will live in `scripts/` once server is built.
